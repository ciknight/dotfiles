ZSH Notes Here
==============

* I'm using the [pure](https://github.com/sindresorhus/pure) prompt
    * I used the [oh-my-zsh integration](https://github.com/sindresorhus/pure#oh-my-zsh) option (so this zsh folder doesn't totally recreate my system.
    * [zsh-syntax-highlighting](https://github.com/zsh-users/zsh-syntax-highlighting) was recommended for this prompt.
    * I also installed the [Tomorrow Night Eighties](https://github.com/chriskempson/tomorrow-theme) theme using [these instructions](http://iterm2colorschemes.com/).
    * Use the [Droid Sans Mono](http://www.1001freefonts.com/search.php?q=Droid+Sans+Mono&search=search) font (install using osx FontBook app).
